# ADR-0005: MCP as first-class surface

## Status

Accepted

## Context

Agent integrations require stable, structured, policy-aware resource access.

## Decision

Treat MCP as a first-class interface, not as an afterthought wrapper.

## Consequences

- Positive: Better agent interoperability.
- Negative: Additional contract maintenance.
- Follow-up: Align MCP resources with canonical IDs and policy filters.
