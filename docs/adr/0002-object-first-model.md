# ADR-0002: Object-first model

## Status

Accepted

## Context

Page-centric implementations lose structure and verification traceability.

## Decision

Model Entity/Claim/Evidence as first-class objects; pages are compiled projections.

## Consequences

- Positive: Better traceability and regeneration.
- Negative: Higher initial modeling effort.
- Follow-up: Keep projection rules explicit in artifact components.
