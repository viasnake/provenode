# Glossary (v0.1)

- Entity: A real-world or logical object represented in the knowledge base.
- Claim: A machine-handleable statement about an entity or relationship.
- Evidence: A traceable source that supports a claim.
- Artifact: A compiled output for human or agent consumption.
- Procedure: Executable operational knowledge with ordered steps.
- Command: Reusable command-level operational primitive.
- Relationship: Explicit edge between knowledge objects.
- Freshness: A class indicating how recently a claim was verified.
- Conflict: A state indicating contradiction or competing interpretations.
