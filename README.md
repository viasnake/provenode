# Provenode

Tagline: Evidence-first knowledge operating system for humans and AI agents.

## Offering policy
- OSS: single-user and small-team self-hosted mode
- Managed: multi-tenant governed mode

## Initial target users
1. Individual builders
2. Small engineering teams
3. Enterprise platform teams

## Non-goals
- Real-time collaborative editor replacement
- General-purpose note app
- Single-vendor AI runtime lock-in

See docs for architecture and operations.
